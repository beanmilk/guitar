IE 생성한다. 

; 테스트할 ID/PW를 설정 (사용가능한 장보로 수정 변경 필요)
$네이버ID=username 설정한다.
$네이버PW=userpw 설정한다. 

$네이버ID=username 같으면 "네이버 ID/PW 정보를 변경하시고 테스트 진행해 주시기 바랍니다." 로그쓰기 하고, 실패한다.

http://nid.naver.com/nidlogin.logout?returl=http://www.naver.com 접속한다.

; 보안1단계 클릭
"[a:title=보안1단계]"를 클릭한다. , 

; ID 입력
"[input:name=id^id=uid]"를 클릭하고 ,  "^a{DEL}" 키를 누른다.  $네이버ID을 입력 한다.
마우스숨김한다. , 

; PW 입력
"[input:name=pw^id=upw]"를 클릭하고 , $네이버PW를 입력 한다. 

; 로그인 수행
"[input:alt=로그인]"를 클릭한다. 

; 로그인 상태 확인
"[a::로그아웃]"를 확인한다. 

현재화면을 캡쳐한다.

IE 종료한다.
